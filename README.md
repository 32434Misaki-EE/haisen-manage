# Haisen Manage - コンセント・ケーブル管理ツール

2次元キャンバス上でコンセント、延長コード、デバイスを配置し、ケーブルで接続して複雑な配線を視覚化・管理するツールです。

## 🎯 概要

このツールは、電源管理の複雑さを解決するために設計されています。以下のような場面で活躍します：

- 📍 **イベント会場の電源配置図作成**
- 🏢 **オフィスの配線ルート管理**
- 🔌 **スタジオ機材の接続図管理**
- 🎬 **映像スタジオの電源・ケーブルネットワーク**

## ✨ 主な機能

### 📦 ノード管理
- **壁コンセント(Wall Outlet)**: 電力供給源（◉）
- **延長コード(Power Strip)**: 複数の差込口を提供（◉）
- **デバイス(Device)**: 電力消費機器（◉）

各ノードは以下のカスタマイズが可能：
- ✏️ **名前設定**: ノード名をリアルタイム編集
- 🎨 **カラー選択**: 4色から選択可能
- 🖼️ **アイコン変更**: 24種類のMaterial Design アイコンから選択
- 📝 **メモ欄**: 詳細情報や仕様を記入
- 📋 **説明欄**: 長めの説明テキストに対応（リサイズ可能）

### 🔌 ポート管理
- **差込口（Socket）**: ノード下部の出力端子
  - `+` ボタンで動的に追加/削除可能
  - ポート型ドロップダウン（ジャンル＞タイプ）で接続先を指定
  - ラベル付き（最大90文字まで対応）

- **プラグ（Plug）**: ノード上部の入力端子
  - 同様に動的に追加/削除可能
  - メモ欄で接続情報を記録

### 🔗 ケーブル接続
- **ドラッグ&ドロップ接続**: 差込口またはプラグからドラッグして接続
- **双方向接続**: どちらから開始しても接続可能
- **自動パス描画**: 曲線を使った視認性の高いケーブル表示
- **クリック削除**: ケーブルをクリックして接続を削除

### 🎮 ワークスペース操作
- **パンニング**: ワークスペース上をドラッグして移動
- **ズーム**: マウスホイール（Ctrl+）で50%-200%の範囲でズーム
- **グリッド吸着**: ノード移動・リサイズ時に20pxグリッドで自動吸着

### 💾 保存・復元
- **自動保存**: 変更は250msのデバウンス後に自動保存
- **localStorage対応**: ブラウザのローカルストレージに保存
- **インポート/エクスポート**: JSON形式で全データをバックアップ・復元

## 🚀 クイックスタート

1. **index.htmlをブラウザで開く**
   ```
   file:///path/to/index.html
   ```

2. **ノードを追加する**
   - 左ツールバーのボタンをクリック

3. **ケーブルを接続する**
   - 差込口（◯）をドラッグして対象ノードのプラグにドロップ

4. **ワークスペースを操作**
   - 背景をドラッグしてパン
   - Ctrl+ホイールでズーム

## 📁 プロジェクト構成

```
haisen/
├── index.html              # メインHTMLテンプレート
├── styles.css              # スタイル定義
├── config.js               # グローバル変数・定数
├── utils.js                # ユーティリティ関数
├── cable.js                # ケーブル接続管理
├── port.js                 # ポート型ドロップダウン管理
├── node.js                 # ノード操作・管理
├── ui.js                   # UI操作（色・アイコン選択）
├── zoom-pan.js             # ズーム・パンニング機能
├── storage.js              # localStorage管理
├── main.js                 # 初期化・イベント設定
├── port-data.js            # ポート型データ定義
├── port.json               # ポート型マスターデータ
└── svgs/                   # アイコンSVGファイル（24個）
```
