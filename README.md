# コンセント負荷マップ

2次元キャンバス上でコンセントや延長コードを配置し、ケーブルで接続して負荷をシミュレーションするツールです。

## 使い方

1. `index.html` をブラウザで開きます。
2. ツールバーのボタンをクリックして要素を追加します。
    - **壁コンセント(Wall Outlet)**: 電力の供給源です。
    - **延長コード(Power Strip)**: 差込口を拡張します。
    - **デバイス(Device)**: 電力を消費する機器です。W数を設定できます。
3. **接続**: 
    - 「差込口（◯）」からドラッグして、「プラグ（上の入力端子）」にドロップすると接続できます。
    - 逆に「プラグ」からドラッグして「差込口」にドロップすることも可能です。
4. **移動**: ノードのヘッダー部分をドラッグして自由に配置できます。
5. **負荷計算**: 
    - 接続状況に応じて負荷が自動計算されます。
    - コンセントや延長コードの許容量を超えると赤く点滅して警告されます。
